# 💳 Credit Default Prediction Using Python

A complete Machine Learning and Data Analysis project that predicts whether a customer is likely to default on a loan using financial information. The project includes data preprocessing, Exploratory Data Analysis (EDA), visualization, feature analysis, and predictive modeling.

---

# 📌 Project Overview

Financial institutions need accurate models to identify customers who are likely to default on loans. This project analyzes customer financial data and builds a predictive model that helps estimate default risk.

---

# 🎯 Objectives

- Import and preprocess financial data
- Detect missing values and duplicates
- Explore customer financial information
- Visualize important trends
- Analyze relationships among variables
- Build a Machine Learning classification model
- Evaluate prediction performance

---

# 📂 Dataset

Dataset used:

**finance.xlsx**

The dataset contains approximately **10,000 customer records** including:

- Income
- Balance
- Student Status
- Default Status
- Age
- Other financial attributes
